# 💰 Expense Tracker (CLI Project)

## 📌 Description
A simple command-line application to track daily expenses.  
Users can add expenses, view total spending, and see category-wise breakdown.

---

## 🚀 Features
- Add expense (amount + category)
- Show total expense
- Show category-wise expense
- View all expenses
- Data stored in a file (expenses.txt)

---

## 🛠️ Technologies Used
- Python
- OOP (Classes)
- File Handling
- Hashmap (Dictionary)
- Prefix Sum (running total concept)

---

## ▶️ How to Run
1. Open terminal in project folder  
2. Run: python main.py
